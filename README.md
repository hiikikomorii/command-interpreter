## Скрипт
**На данный момент ``(19.12.2025)`` скрипт имеет ``6 функций``**
1. **``Exit`` - Выход из скрипта**
2. **``Clear`` - Очистка консоли**
3. **``Help`` - Выводит все доступные команды**
4. **``IP``- Показывает ваш ``локальный ip``**
5. **``Info`` - Выводит инфомацию о системе**
6. **``Time`` - Показывает текущуюю ``дату``, ``время``, и ``день недели``**

**Так же, скрипт имеет в коде строку ``ConsoleColor.Blue;``, что делает все символы синим цветом**

**Вы можете поменять текущий цвет на ваш любимый, изменив ``12`` и ``20`` строки.**

## Установка
**Dotnet version - ``[.NET 8.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)**

**Нужно скачать папку ``scr``, затем извлечь ее из zip-архива, после чего открываем ``cmd`` и вписываем следующие команды:**
``
cd "путь к ConsoleApp3.csproj"
dotnet run``

**После этих действий, скрипт должен запуститься.**
